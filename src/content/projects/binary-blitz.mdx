Binary Blitz is a small game built using React and written in TypeScript designed to help
people learn how to convert binary values to decimals through repetition. As the game
progresses, it becomes incrementally harder to solve.

Check out the <ExternalLink href="https://github.com/stephenlunt/binary-blitz" text="GitHub repo here" />.
