Scrabble Checker (<a href="https://scrabblechecker.com" target="_blank" rel="noopener" class="inline-flex items-center gap-1">scrabblechecker.com <BiLinkExternal /></a>) is an online word-finding tool I built for Scrabble
players to learn more about Scrabble and words from the dictionary.

It primarily runs on the Django framework on the backend which handles most of the application
logic server side. Vanilla JavaScript is used on the frontend to add interactivity for users.
Finally, Docker is used to containerise the Django application, including the Postgres
database, nginx proxy, and Redis cache, which makes for easy deployment.
